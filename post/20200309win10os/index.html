装系统风波。

系统
2020-03-10

隔离回公司第一件事就是装系统

说来奇怪
因为疫情事件公司电脑一个多月木有开机，结果一启动多台电脑无法开机，
甚至系统无法恢复，一直循环在自修状态并维持一上午。
嘿，更惨的是还没带U盘，上班的同事还都没拿，也就弄不了PE；

第一步借用其他主机借尸还魂

把固态硬盘拆了直接放在好的上装完换回，其实此方法是可行的，可惜是做了件错事把盘隔了没分好区。
这是后话了，装完的我直接拆了过去，怎么开都开不开，一度以为主板设置问题，不停扣boot
期间还遇到了
windows安装程序无法将windows配置为此计算机的硬件上运行。微软官方给出：

手动运行 msoobe 以完成安装
此问题可能的解决方法是手动运行 msoobe 程序 (这可确保激活并正确注册 Microsoft Windows 版本) 以允许安装完成。请按照以下步骤手动运行 msoobe 程序:
Windows 7:
在错误屏幕中, 按Shift + F10打开命令提示符 (或在 Windows 搜索栏中键入cmd , 并从搜索结果菜单中选择 "命令提示符")。
键入cd \ , 然后按 enter 键。
键入cd x: \windows\system32\oobe ( x是安装 Windows 的驱动器号, 例如 c:\windows\system32\oobe), 然后按 enter 键。
键入msoobe , 然后按 enter 键。安装过程现在应该会自动继续。
卸下安装介质, 系统应完成安装并引导至 Windows。
Windows 10:
在显示错误的屏幕中, 按Shift + F10以打开命令提示符。
键入CD x: \windows\system32\oobe (其中x是安装了 Windows 的驱动器号, 例如 C:\windows\system32\oobe), 然后按Enter 键。
键入msoobe , 然后按Enter 键。
然后, 系统会提示您创建帐户名称和密码, 并设置时间和日期。完成后, 单击完成。
注: 如果这是 Windows 10 的零售版本, 还可能会提示您输入 Windows 10 的产品密钥。输入产品密钥, 然后单击Finish (完成)。
安装过程随后应完成, 并允许计算机引导至 Windows。

然并卵第二步还得靠UP

找到了年前在张大妈上捡的雷克沙，随便抓了个pe系统就上了，进去一看才发现分区问题，没给系统盘分ESP MSR

https://jingyan.baidu.com/article/4f34706e7daeb6e387b56d00.html 关于esp/msr分区方面

之后的步骤就很顺畅了，完美结局，但是无缘无故怎么就都系统出了问题，还有一个固态+机械，单飞固态就完美，双剑合璧就卡逼，还是都换了固态可行～

总结

没有pe装不了的系统
分区一定要看好
