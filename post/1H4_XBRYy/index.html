纪录更换服务:Windows 2008 r2 换 Centos

2019-04-13 服务器

思路

备份win所有文件，因为是全盘格式化

完整下载到本地，并成功在虚拟机搭建，以备出错

搭建centos 环境，主用宝塔面板 朋友推荐，为了方便

宝塔确实好用，基本界面操作，该有的都有了 注意之后网站维护安全方面就行

备份

直接把服务器所有文件及数据库打包 丢到网站根目录迅雷下载速读取决于服务器的网速 下载要写全文件名比如：www.www.com/sss/sss.zip要不然可能出现403 访问错误

本地搭建

虚拟机用的Oracle VM VirtualBox 下载地址就是官网 https://www.virtualbox.org/wiki/Downloads

下载centos 7.6 http://mirrors.163.com/centos/7.6.1810/isos/x86_64/CentOS-7-x86_64-Minimal-1810.iso 遇到问题 不知道是不是笔记本wifi的问题，桥接网络一直连不上 网卡自动无法开启 可以先用 ip add 查看是否开启网络 或者ping 一下

打开网卡命令： ifup enp0s3

正常开启网络 就可以安装宝塔了

安装套件

相应的配置版本 按照之前的版本就可以 编译极速看需求，本地选的极速 服务器选的编译，速读快和慢，另外就是稳定性

数据库这里PHPMyadmin出现了问题，文件超过50m 改了php.ini 上传大小和响应时间，均是无法上传： 报错： 502 Bad Gateway

然后弃用phpmyadmin改用shh命令行 进入数据库： mysql -uroot -h127.0.0.1 -p 看一下有哪些库 show databases; 进入库 use sss库名 上传数据库文件 source 文件路径

添加网站

因为我的数据库使用php直接调用所以新建不选建数据库 根目录和域名写对其他默认就可以

因为我是本地搭建线上网站，所以域名添加之后要去更改host文件 打开方式可以在搜索中搜索记事本右键管理员运行 要不然没发修改 C:\Windows\System32\drivers\etc 因为是虚拟机所以ip一定要写虚拟机ip 切勿写127.0.0.1 搞定到这基本上就可以了，其他的就是调试了，没问题就可以直接上手安装服务器了；

安装后 宝塔需要更改一些内容以便增加服务器安全

面板设置
面板端口：默认全是8888 非常不安全   8888 - 65535之间随便
安全入口：开始是随机的，虽然很安全，但是不好记啊，改个自己能记住也不要太短的
域名：这个我加上了，用ip总觉得不得劲，另外域名设置好之后只能用过域名访问，ip就用不了了，也算是增加了安全吧
IP：这个就算了 除了固定ip都不要去设置这个，动态ip换了就没法登录了
面板用户：改
面板密码：改
FTP端口：改了
数据库：回收站要记得开
开启监控：已打开，方便查看负载纪录
安全：ssh是打开了，之前看到一个帖子说是不建议打开：因为面板已经安装好，后期基本用不到。这个端口很容易被扫描。但是端口是必须要改的
PING：这个随意，我关掉了，关掉ping对seo没什么影响，只是在黑客在扫ip的时候ping不通的可能就略过了，另外关了这玩意自己也ping不通，有适合测试网站的时候也有影响
软件安装：
高防IP，
七牛云存储，这个还在申请中，据说10g免费，七天备份很方便，
PHP守护：
监控PHP-FPM运行状态，防止大批量出现502错误!````
